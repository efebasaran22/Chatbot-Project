# 🤖 Üniversite Chatbot ve Veri Analizi Projesi

Bu proje, Python kullanılarak geliştirilmiş bir chatbot uygulamasının arka plan mimarisini ve veri işleme süreçlerini içermektedir.

### 🚀 Proje Hakkında
Bu çalışma, üniversite ortamında yürütülen bir grup projesi olup, chatbot'un öğrenme verimliliğini artırmak için verilerin analiz edilmesi ve yapılandırılmasına odaklanmaktadır.

* **Ekip Çalışması:** Proje yönetimi ve görev dağılımı **Azure DevOps** platformu üzerinden Çevik (Agile) metodolojilerle yürütülmüştür.
* **Veri Analizi:** Gelen ham verilerin chatbot mimarisine uygun hale getirilmesi ve anlamlandırılması.
* **Modüler Yapı:** API, servisler ve veri (data) katmanlarının ayrı ayrı kurgulandığı modüler Python mimarisi.

### 🛠️ Kullanılan Teknolojiler
* **Programlama Dili:** Python
* **Proje Yönetimi:** Azure DevOps, Git
* **Mimari:** Modüler API ve Servis Yapısı
